# Number System Converter

Program that convert number between decimal, binary, octal and hexadecimal.

## Sample execution of the program

```exection
Number System Converter. Please select option [1-4]
[1] Decimal
[2] Binary
[3] Octal
[4] Hexadecimal
1
Enter a number: 100
Binary: 1100100
Octal: 144
Hexadecimal: 64
Try again? [Y/N]
```
